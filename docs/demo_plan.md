# 🎥 JarvisAi Hackathon Demo Plan (2 Minutes)

**Goal:** Demonstrate speed, "Focus Mode" (the killer feature), and seamless macOS integration.

**Pre-requisites:**
1.  Run `python main.py` in a terminal.
2.  Have Chrome open with a few "safe" tabs (GitHub, Google Docs).
3.  Ensure volume is up to hear Jarvis's voice.
4.  **Important:** Ensure you have granted "Accessibility" permissions to your terminal if prompted.

---

## 🎬 Act 1: The Hook & Speed (0:00 - 0:30)

*   **0:00 - 0:10 | Intro**
    *   **Visual:** User sitting at desk, hands clearly *away* from keyboard/mouse. Screen shows a clean desktop with the terminal running.
    *   **Action:** User says: *"Jarvis."*
    *   **Effect:** The **Blue Pulse Overlay** appears instantly around the screen.
    *   **Voiceover:** "This is Jarvis. Not just a chatbot, but a flow state guardian built for speed."

*   **0:10 - 0:30 | Lightning Navigation**
    *   **Action:** User says: *"Jarvis, open GitHub."*
    *   **Effect:** Chrome instantly opens/focuses GitHub. Overlay pulses and fades.
    *   **Action:** User says: *"Jarvis, switch to Google Docs."*
    *   **Effect:** Instant switch.
    *   **Action:** User says: *"Jarvis, close this tab."*
    *   **Effect:** Tab closes.
    *   **Key Point:** Emphasize that there is **zero latency**. It feels native.

## 🎬 Act 2: The Killer Feature - "Focus Mode" (0:30 - 1:15)

*   **0:30 - 0:45 | Setting the Trap**
    *   **Action:** User says: *"Jarvis, focus on coding."*
    *   **Effect:** Jarvis speaks: *"Focus mode on. General productivity."*
    *   **Voiceover:** "Now, Jarvis is watching. If I try to procrastinate..."

*   **0:45 - 1:00 | The Violation**
    *   **Action:** User manually opens a new tab and types `youtube.com`.
    *   **Effect:** 
        1.  Video starts loading.
        2.  **Jarvis speaks:** *"Stay focused."* (Warning)
        3.  User ignores it (wait 3 seconds).
        4.  **Action:** Tab **AUTOMATICALLY CLOSES**.
        5.  **Jarvis speaks:** *"Closing distraction."*

*   **1:00 - 1:15 | The "No-Mercy" Re-test**
    *   **Action:** User tries to open `twitter.com`.
    *   **Effect:** Warning... then closed again.
    *   **Voiceover:** "It ruthlessly defends your attention span."

## 🎬 Act 3: Rescue & Conclusion (1:15 - 2:00)

*   **1:15 - 1:35 | Restoration**
    *   **Action:** User says: *"Jarvis, break time."*
    *   **Effect:** Jarvis: *"Break time."* (Monitoring disabled).
    *   **Action:** User says: *"Restore my tabs."*
    *   **Effect:** The closed YouTube/Twitter tabs re-open.
    *   **Voiceover:** "But it's forgiving when you need it to be."

*   **1:35 - 2:00 | Outro**
    *   **Action:** Show the "Blue Wave" overlay one last time.
    *   **Voiceover:** "Built with Faster-Whisper, Gemini Flash, and native AppleScript. This is Jarvis. Local, private, and fast."
    *   **Action:** User says: *"Jarvis, exit."*
    *   **Fade to Black.**
